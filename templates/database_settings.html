Environment Variables Configuration

To modify database configurations, update the following environment variables:

Local Database:
  • LOCAL_SERVER
  • LOCAL_DATABASE
  • LOCAL_UID
  • LOCAL_PWD
Azure Database:
  • AZURE_SERVER
  • AZURE_DATABASE
  • AZURE_UID
  • AZURE_PWD

Active Database: DATABASE_TYPE (set to 'local' or 'azure')
